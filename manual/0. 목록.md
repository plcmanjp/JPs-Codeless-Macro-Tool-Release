# [사용자 매뉴얼] 0. 목록: JP's Codeless Macro Tool 사용법 전체 가이드

## JP's Codeless Macro Tool 사용자 매뉴얼 목록

이 문서는 사용자 매뉴얼의 전체 목차입니다.
처음 사용하는 경우에는 번호 순서대로 읽는 것을 권장합니다.

## 빠른 시작

처음이라면 아래 순서로 읽으면 됩니다.

1. **기능설명**에서 프로그램으로 무엇을 할 수 있는지 확인합니다.
2. **기본 편집과 파일관리**에서 프로젝트 생성, 저장, 스텝 편집 방법을 익힙니다.
3. **녹화와 재생**에서 실제 동작을 기록하고 실행하는 방법을 확인합니다.
4. 필요한 기능에 따라 조건, 반복, 포인터, 변수, 정규식 추출, 이미지 검색, 샘플 스텝, 즐겨찾기 문서를 참고합니다.
