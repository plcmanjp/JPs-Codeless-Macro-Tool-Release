# JP's Codeless Macro Tool v1.3.0 업데이트

출시일: (v1.3.0 배포 시 확정)

게임·웹·일반 프로그램에서 입력 호환성을 크게 높이고, 프로젝트 이동·공유 등 여러 동작을 안정화한 업데이트입니다.

## 새로운 기능

- **게임·프로그램 키 입력 호환성 강화(스캔코드)**: 일부 게임이나 프로그램이 일반 키 입력을 인식하지 못하던 문제를 개선했습니다. 표준 Windows 입력 방식(스캔코드)을 우선 사용하여, 기존에 키 입력이 동작하지 않던 환경에서도 매크로가 작동합니다. 일반 프로그램 입력은 이전과 동일합니다.

- **상대 마우스 이동(FPS 시점 회전)**: 마우스 동작에 "상대 이동" 옵션을 추가했습니다. 절대 좌표가 아닌 상대 이동량(Δx, Δy)으로 움직여 FPS 게임의 시점 회전 등에 사용할 수 있습니다.

- **백그라운드(비활성 창) 입력**: 매크로를 보낼 대상 창을 지정하면, 그 창이 활성화되지 않은 상태에서도 입력을 보낼 수 있습니다. 대상 창은 화면에서 직접 클릭해 선택하며(선택할 때 화면이 어두워지고 고른 창만 밝게 강조됩니다), 표준 Windows 프로그램에서 동작합니다. (일부 게임은 자체 입력 처리 방식으로 인해 동작하지 않을 수 있습니다.)

- **권한 안내**: 대상 창이 관리자 권한으로 실행 중이라 입력이 막힐 수 있는 경우 미리 안내합니다.

## 개선 사항

- **게임 클릭 발사 안정화**: 마우스 클릭이 일부 게임에서 발사로 인식되지 않던 문제를 개선했습니다(누르고 떼는 사이 간격 확보).

- **프로젝트 이동·공유 호환성**: 이미지 조건에 등록한 이미지(여러 장)와 무시영역 마스크의 경로를 프로젝트 폴더 기준으로 저장해, 폴더를 옮기거나 다른 PC로 복사해도 이미지 검색이 정상 동작합니다. 이전 버전에서 만든 프로젝트도 열기만 하면 자동으로 경로가 복구됩니다.

- **즐겨찾기 순서 변경 안정화**: 즐겨찾기 관리 창에서 드래그로 순서를 바꾼 뒤 확인하면, 선택한 항목이 정확히 반영되고 바뀐 순서도 그대로 저장됩니다.

- **변수 표시 안정성**: 16진수·8진수 변수에 문자값이 들어간 상태로 텍스트 입력에 사용해도 매크로가 중단되지 않습니다.

- **변수 계산(CALC) 보호 강화**: 지나치게 큰 거듭제곱 수식이 프로그램을 멈추게 하던 문제를 차단했습니다.

- **프로젝트 이름 검사**: 새 프로젝트·다른 이름으로 저장·이름바꾸기에서 폴더 이름에 쓸 수 없는 문자를 입력하면 안내합니다.

- **종료 안정성**: 매크로 재생 중에 프로그램을 닫아도 안전하게 종료됩니다.

## 안내

이 기능들은 표준 Windows 입력/캡처 경로를 사용하며, 게임 탐지 회피나 안티치트 우회를 위한 것이 아닙니다.
