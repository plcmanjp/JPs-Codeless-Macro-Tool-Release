# JP's Codeless Macro Tool v1.0.40 업데이트

## 새로운 기능

- **로그 모니터 창**: 매크로 실행 동작을 실시간으로 보여주는 로그 모니터 창을 추가했습니다. 메인 창 오른쪽에 변수 모니터와 함께 나란히 붙고, 순서를 바꾸거나 따로 떼어 놓을 수 있습니다.
- **로그 파일 자동 기록**: 키보드 입력, 마우스 이동·클릭, 이미지 검색 결과 등 주요 동작이 로그 파일로 자동 기록됩니다. 옵션 변경, 재생, 저장, 파일 열기 시에도 기록이 남습니다.
- **항상 위 옵션**: 메인 창과 보조 창을 항상 다른 창 위에 띄우는 "항상 위" 옵션을 추가했습니다.
- **로그 레벨 설정**: 옵션 창에서 로그 상세 수준(기본/DEBUG/INFO/WARN/ERROR)을 선택할 수 있으며, 바꾸면 재시작 없이 즉시 적용됩니다.
- **좌표 어레이 화면 표시**: 좌표 어레이를 편집할 때 등록한 좌표를 화면 위에 번호와 빨간 점으로 표시해 위치를 바로 확인할 수 있습니다.

## 개선 사항

- 이미지 캡처 영역을 마우스로 옮기거나 크기를 조절할 수 있습니다. 선택 영역을 확대 미리보기 팝업으로 확인할 수 있습니다.

## 버그 수정

- 이미지를 캡처할 때 선택 영역의 왼쪽·위쪽 1픽셀이 잘리던 문제를 수정했습니다.
