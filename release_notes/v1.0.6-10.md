# JP's Codeless Macro Tool v1.0.6~v1.0.10 업데이트

v1.0.5 이후 안정성 개선과 버그 수정 위주의 연속 패치입니다.

## 개선 사항 (v1.0.7~v1.0.10)

- 포인터로 연결된 스텝이 더 안정적으로 함께 관리되도록 개선했습니다.
- 이미지 조건 스텝의 이름을 변경할 수 있게 했습니다.
- 여러 스텝을 선택한 뒤 위/아래로 이동할 때 선택이 유지되도록 개선했습니다. 이동 단축키도 더 안전한 조합으로 변경했습니다.
- 프로그램 내부 구조를 기능별로 나누어 이후 기능 추가와 유지보수가 쉬워졌습니다. 사용자 기능은 기존과 동일합니다.

## 버그 수정

- 일부 환경에서 텍스트 입력 스텝을 열 때 오류가 나던 문제를 수정했습니다. (v1.0.6)
- 포인터 관련 주석이 사라지던 문제를 수정했습니다. (v1.0.7)
- 듀얼 모니터에서 이미지 검색 좌표가 잘못 처리되던 문제를 수정했습니다. (v1.0.8)
