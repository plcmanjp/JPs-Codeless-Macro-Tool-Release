# JP's Codeless Macro Tool v1.0.44 업데이트

출시일: 2026-05-31

## 새로운 기능

- **후원하기**: 이 프로그램은 모든 기능을 무료로 제공합니다. 도구가 도움이 되셨다면 개발을 응원할 수 있도록 메인 화면 툴바와 도움말에 "후원하기" 버튼을 추가했습니다.
  - 버튼을 누르면 카카오페이·토스·페이팔 QR이 표시되어 원하는 방법으로 후원할 수 있습니다.
  - 후원은 전적으로 선택 사항이며, 모든 기능은 변함없이 무료로 사용할 수 있습니다.
- **업데이트 변경 로그 안내**: 새 버전으로 업데이트한 뒤 처음 실행하면 무엇이 달라졌는지 알려주는 안내 창이 한 번 표시됩니다.
  - 최신 두 개 버전의 변경점을 함께 보여 주며, 같은 버전에서는 다시 표시되지 않습니다.
  - "자세히 보기"로 이 배포 글을 바로 열 수 있습니다.
