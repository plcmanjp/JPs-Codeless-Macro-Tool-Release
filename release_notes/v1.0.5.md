# JP's Codeless Macro Tool v1.0.5 업데이트

출시일: 2026-05-11

## 새로운 기능

- **포인터 호출**: 특정 스텝 묶음을 여러 곳에서 재사용할 수 있는 포인터 호출 기능을 추가했습니다.
- **정규식 추출**: 변수에 저장된 문자열에서 필요한 값만 정규식으로 찾아 다른 변수에 저장하는 스텝을 추가했습니다. 편집 화면에서 원본 텍스트와 패턴을 미리 테스트할 수 있습니다.
- **업데이트 알림**: 새 버전이 있을 때 자동으로 알려주는 기능을 추가했습니다.

## 개선 사항

- 스크립트 프로젝트 저장 경로를 더 관리하기 쉽게 정리했습니다.
