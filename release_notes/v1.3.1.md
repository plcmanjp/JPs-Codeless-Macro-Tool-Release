# JP's Codeless Macro Tool v1.3.1 업데이트

출시일: (v1.3.1 배포 시 확정)

백그라운드 입력의 정확도를 높이고 창 선택 사용성을 개선한 업데이트입니다.

## 개선 사항

- **백그라운드 입력 좌표를 대상 창 기준으로 관리**: 백그라운드 대상 창에 마우스 입력을 보낼 때, 좌표를 그 창 내부 위치 기준으로 저장합니다. 대상 창을 화면에서 옮기거나 위치가 달라져도 창 안의 같은 지점을 정확히 클릭합니다. (대상 창을 지정하지 않은 기존 매크로는 이전과 동일하게 동작합니다.)

- **대상 창을 찾지 못할 때 안내**: 좌표를 캡처(F2)하거나 녹화를 메인에 추가할 때 지정한 대상 창을 찾지 못하면, 잘못된 위치가 저장되지 않도록 안내하고 작업을 멈춥니다. 대상 창을 실행한 뒤 다시 시도하면 됩니다. (녹화 내용은 그대로 유지됩니다.)

- **창 선택 화면 정리**: 백그라운드 대상 창을 화면에서 직접 클릭해 고를 때 뜨는 선택 화면이 작업표시줄에 표시되지 않도록 정리했습니다.

## 안내

이 기능들은 표준 Windows 입력 경로를 사용하며, 게임 탐지 회피나 안티치트 우회를 위한 것이 아닙니다.
