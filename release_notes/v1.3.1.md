# JP's Codeless Macro Tool v1.3.1 업데이트

출시일: 2026-06-15

게임·웹·일반 프로그램 입력 호환성을 강화하고, 백그라운드 입력·전체화면 화면 인식·마우스 상대 이동 등 새 기능을 추가한 업데이트입니다.

## 새로운 기능

- **게임·웹·일반 프로그램 입력 호환성 강화**: 일부 프로그램이 인식하지 못하던 키 입력을 표준 경로(스캔코드)로 보강하고, 마우스 상대 이동(시점 회전 등)을 지원합니다.

- **백그라운드 입력**: 대상 창을 맨 앞으로 가져오지 않고도 그 창에 마우스·키 입력을 보냅니다. 좌표는 대상 창 내부 위치 기준이라 창을 옮겨도 같은 지점을 정확히 클릭합니다.

- **전체화면 게임 화면 인식(DXGI 캡처)**: DirectX 전체화면 게임에서 화면이 검게 캡처되어 이미지 인식·OCR이 실패하던 문제를 해결했습니다.

- **마우스 상대 좌표 이동 클릭**: 클릭·누름·뗌에 "상대 좌표 이동(Δx/Δy)" 옵션을 추가했습니다. FPS 시점 회전 후 발사처럼 상대 이동과 버튼 동작을 한 스텝으로 처리하며, "절대 좌표 이동"과 "상대 좌표 이동" 중 하나만 선택합니다. 상대 이동에도 이동 시간을 적용해 회전 속도를 조절할 수 있습니다.

- **F2 빠른 캡처 화면 표시**: 메인에서 F2로 좌표를 등록할 때 커서 위치에 십자선과 좌표를 잠깐 표시해 등록 지점을 바로 확인할 수 있습니다.

## 개선 사항

- **백그라운드 입력 좌표를 대상 창 기준으로 관리**: 백그라운드 대상 창에 마우스 입력을 보낼 때, 좌표를 그 창 내부 위치 기준으로 저장합니다. 대상 창을 화면에서 옮기거나 위치가 달라져도 창 안의 같은 지점을 정확히 클릭합니다. (대상 창을 지정하지 않은 기존 매크로는 이전과 동일하게 동작합니다.)

- **대상 창을 찾지 못할 때 안내**: 좌표를 캡처(F2)하거나 녹화를 메인에 추가할 때 지정한 대상 창을 찾지 못하면, 잘못된 위치가 저장되지 않도록 안내하고 작업을 멈춥니다. 대상 창을 실행한 뒤 다시 시도하면 됩니다. (녹화 내용은 그대로 유지됩니다.)

- **창 선택 화면 정리**: 백그라운드 대상 창을 화면에서 직접 클릭해 고를 때 뜨는 선택 화면이 작업표시줄에 표시되지 않도록 정리했습니다.

## 안내

이 기능들은 표준 Windows 입력 경로를 사용하며, 게임 탐지 회피나 안티치트 우회를 위한 것이 아닙니다.
