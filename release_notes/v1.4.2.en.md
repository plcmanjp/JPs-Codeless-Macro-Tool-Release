# v1.4.2

## Improvements

- **Lighter runtime dependencies**: Removes an unnecessary external input-library dependency from macro playback, simplifying the packaged app and runtime environment.
- **Preserved active-input behavior**: Keeps the existing behavior for mouse movement, clicks, dragging, scrolling, key input, and text input while cleaning up the internal input path.
- **More reliable coordinate capture and tests**: Updates coordinate capture and regression tests to use the new input path, making future maintenance and verification clearer.
