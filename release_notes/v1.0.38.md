# JP's Codeless Macro Tool v1.0.38 업데이트

출시일: 2026-05-21

## 새로운 기능

- **ELSE IF 조건 분기**: IF → ELSE IF → ELSE IF → ELSE 순서로 여러 조건을 이어 붙여 원하는 분기만 실행할 수 있습니다.
- **조건 스텝 타입 전환**: 조건 스텝(AND / OR / ELSE / ELSE IF)을 더블클릭하면 다른 타입으로 바꿀 수 있습니다.

## 개선 사항

- 조건 체인의 표시 이름이 한글로 바뀌었습니다. AND → 그리고, OR → 또는, ELSE → 아니면, ELSE IF → 아니면 만약.

## 버그 수정

- 조건 종류를 전환할 때(예: 이미지 발견됨 → 항상 참) 이전 위젯 참조로 인해 발생하던 오류를 수정했습니다.
