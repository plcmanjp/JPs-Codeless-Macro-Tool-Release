# JP's Codeless Macro Tool v1.0.19~v1.0.20 업데이트

## 새로운 기능 (v1.0.20)

- **변수 초기화 선택**: 반복 실행 시 변수마다 초기화 여부를 선택할 수 있습니다. 어떤 변수는 반복할 때마다 초기값으로 돌리고, 어떤 변수는 이전 값을 유지하도록 설정할 수 있습니다.

## 개선 사항 (v1.0.19)

- 변수 계산에서 8진수 표시 방식을 선택할 수 있게 했습니다.
- 계산식에 쓰는 값과 화면에 보여주는 값의 처리 방식을 더 안정적으로 분리했습니다.
- 변수 설정 화면을 보기 쉽게 정리했습니다.

## 버그 수정

- 16진수 변수 계산이 특정 값 이후 잘못 처리되던 문제를 수정했습니다. (v1.0.19)
- 반복 재생할 때 이전 회차의 변수 값이 다음 회차에 영향을 주던 문제를 수정했습니다. (v1.0.19)
- 초기화하지 않도록 설정한 변수도 반복 재생 때 초기화되던 문제를 수정했습니다. (v1.0.20)
- 변수 비교 조건이 저장된 값과 실행 값이 달라 정상 재생되지 않던 문제를 수정했습니다. (v1.0.20)
