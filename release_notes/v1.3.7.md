# v1.3.7

## 개선 사항

- **재생 안정성**: 매크로를 정지(F10)한 직후 곧바로 다시 재생할 때 드물게 발생할 수 있던 오류와 입력이 두 번 들어가는 문제를 방지했습니다.
- **저장 데이터 보호**: 다른 이름으로 저장 시 이미 있는 프로젝트를 실수로 덮어쓰지 않도록 확인 창을 추가하고, 저장 도중 문제가 생겨도 기존 파일이 손상되지 않도록 안전하게 저장하도록 개선했습니다.
- **백그라운드 입력 정확도**: 백그라운드 대상 창으로 스크롤을 전달하도록 지원하고, 이미지·글자 위치를 기준으로 한 클릭이 정확한 지점에 전달되도록 좌표 처리를 바로잡았습니다.
- **재생 견고성**: 숫자 변수의 정규식 추출, 클립보드 일시 잠금 등 특정 상황에서 매크로가 통째로 중단되던 문제를 방어했습니다.
- **입력 검증 일관성**: 변수 바꾸기·이미지 조건 등에서 필수 항목이 비어 있으면 저장 전에 안내하도록 하고, 변수 진법(10/16/8진수)을 전환할 때 값이 손상되지 않도록 개선했습니다.
- **마우스 동작 편집 개선**: 새 마우스 동작을 추가할 때도 '상대 좌표 이동' 옵션이 바로 표시되고, 스크롤 동작에 스크롤할 위치 좌표를 지정할 수 있도록 개선했습니다.
- **변수 바꾸기 입력**: 검색·바꿀 문자 칸에 `%` 기호를 변수 참조 없이 그대로(리터럴) 입력할 수 있도록 했습니다.
- **세부 안정성**: 여러 예외 상황과 UI 조작에 대한 방어를 폭넓게 보강했습니다.
