# JP's Codeless Macro Tool v1.0.12~v1.0.13 업데이트

출시일: 2026-05-13

## 새로운 기능 (v1.0.12)

- 메인 창과 녹화 창에서 `Delete` 키로 선택한 스텝을 삭제할 수 있습니다. 여러 스텝을 선택한 뒤 한 번에 삭제할 수도 있습니다.

## 개선 사항 (v1.0.13)

- 이미지 검색 기본 정확도를 더 안정적인 값으로 조정했습니다.
- 이미지 검색 결과 표시 테두리를 더 보기 좋게 정리했습니다.
- 새 이미지 검색 전에 이전 표시가 남지 않도록 개선했습니다.
