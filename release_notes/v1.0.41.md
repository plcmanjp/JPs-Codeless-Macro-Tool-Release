# JP's Codeless Macro Tool v1.0.41 업데이트

## 새로운 기능

- **여러 이미지 중 하나라도 보이면 (OR 검색)**: 하나의 이미지 조건에 여러 장을 등록해 그중 하나라도 화면에 보이면 분기하도록 할 수 있습니다.
  - 목록 순서가 검색 우선순위이며, 가장 먼저 발견된 이미지 위치를 클릭 기준으로 사용합니다.
  - "이미지 없음" 조건은 등록한 이미지가 모두 보이지 않을 때만 참이 됩니다.
- **이미지 목록 편집기**: 이미지 조건 편집창이 이미지 목록 + 미리보기 방식으로 바뀌었습니다. 추가(캡처), 편집(다시 캡처·자르기), 이름 변경, 제거, 순서 이동을 지원합니다.
- **기존 이미지 불러오기**: 프로젝트에 저장된 이미지를 선택해 목록에 추가할 수 있습니다.
- **이미지별 개별 설정**: 정확도와 검색 영역(ROI)을 전체 공통값으로 두고, 특정 이미지만 개별 값으로 따로 지정할 수 있습니다.

## 개선 사항

- **조건 결과 표시 유지**: IF 조건의 참/거짓 결과가 재생이 끝난 뒤에도 스텝 목록의 비고에 남습니다. 다음 재생을 시작하면 초기화됩니다.
- **이미지 검색 기록**: 이미지 검색 시 검색 영역과 매칭 점수가 로그에 남아 검색 실패 원인을 확인할 수 있습니다.
- 검색 성공 시 화면에 표시되는 빨간 테두리를 상단 툴바의 "검색결과 표시" 하나로 통일했습니다.

## 버그 수정

- 이미지 위치를 사용하는 마우스 동작이 이전 이미지 검색에 실패한 상태에서 실행될 때 매크로 전체가 멈추던 문제를 수정했습니다. 이제 해당 스텝만 건너뛰고 계속 진행합니다.
- 이미 등록된 이미지 조건 스텝을 편집할 때 편집창이 작게 열리던 문제를 수정했습니다.
