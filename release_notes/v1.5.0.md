# v1.5.0

## 새로운 기능

- **딜레이 시/분/초/ms 분리 입력**: 딜레이(시간 대기) 스텝의 입력창을 시·분·초·ms 네 칸으로 나누었습니다. 각 칸에 입력한 값을 합산한 시간만큼 대기하므로, 긴 대기 시간을 밀리초로 환산해 넣을 필요 없이 직관적으로 지정할 수 있고 수 분 이상의 장시간 대기도 손쉽게 설정할 수 있습니다. 기존에 저장한 매크로의 딜레이 값은 그대로 유지·재생됩니다.

## 개선 사항

- **안정성 및 오류 수정**: 손상된 설정·즐겨찾기 파일에서도 앱이 정상적으로 시작되도록 보강했고, 스텝 복제·드래그·좌표 편집과 백그라운드 입력·이미지 검색 위치 처리에서 발견된 여러 오작동을 수정해 전반적인 안정성을 높였습니다.
