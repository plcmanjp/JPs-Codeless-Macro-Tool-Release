# JP's Codeless Macro Tool v1.0.43 업데이트

출시일: 2026-05-31

## 새로운 기능

- **마우스 좌표 어레이 자동 생성**: 좌표 입력 창에 자동생성 기능을 추가했습니다.
  - 기준 좌표 + X/Y 갯수·피치를 입력하면 격자(또는 가로/세로 1줄) 좌표를 자동으로 생성해 목록에 추가합니다.
  - 채우기 순서 2종(행 우선, 열 우선)을 지원합니다.
  - 피치에 음수를 넣으면 전개 방향이 바뀝니다.
  - "마우스로 피치 계산" 버튼으로 끝점을 찍으면 갯수에 맞춰 피치를 자동 산출합니다.
  - 자동생성 창에서 파라미터를 바꾸면 생성될 격자가 화면에 실시간으로 미리 보입니다.
- **마우스 좌표 입력 UI 테이블 통일**: 단일 좌표 입력과 좌표 어레이를 하나의 인라인 테이블로 통합했습니다.
  - 기본 1행(기존 단일 좌표와 동일 사용감)으로 시작하고, 행을 추가하면 좌표 어레이가 됩니다.
  - 캡처(F2/3초)·행 추가·삭제·전체 삭제·위아래 이동·자동생성을 한 곳에서 처리합니다.
  - 편집 중 좌표를 화면에 번호와 점(반투명)으로 실시간 미리보기하며, 체크박스로 끌 수 있습니다.

## 개선 사항

- 좌표 어레이 화면 인디케이터를 단일 창 일괄 렌더링으로 재작성해, 대량(수백~2,000개) 좌표 표시 지연을 제거했습니다.
