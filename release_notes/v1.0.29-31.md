# JP's Codeless Macro Tool v1.0.29~v1.0.31 업데이트

출시일: 2026-05-16

## 새로운 기능 (v1.0.29)

- **딜레이 클립보드 변경 대기**: 딜레이 스텝에 "시간 대기"와 "클립보드 변경 대기" 유형을 선택할 수 있습니다.
  - 클립보드 변경 대기는 `Ctrl+C` 직후 복사가 완료되는 시점을 기다릴 때 유용합니다.
  - 클립보드가 바뀌면 즉시 다음 스텝으로 진행하고, 지정한 최대 대기 시간이 지나면 변경이 없어도 다음 스텝으로 넘어갑니다.

## 개선 사항 (v1.0.30)

- 마우스 액션 편집창의 좌표 어레이 관련 항목 배치를 정렬하고, 캡처 버튼에 3초 카운트다운 표시를 추가했습니다.

## 버그 수정 (v1.0.31)

- 재생 중 매크로 도구 창에 포커스가 있어도 `Ctrl+C`, `Ctrl+X`, `Ctrl+V`가 스텝 내용을 시스템 클립보드로 복사하거나 편집하지 않도록 차단했습니다.
- 클립보드 변경 대기가 실제 복사 완료가 아닌 경우에 통과되던 문제를 수정했습니다.
