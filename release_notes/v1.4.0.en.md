# v1.4.0

## New Features

- **Now on the Microsoft Store — future updates ship there**: JP's Codeless Macro Tool is now published on the Microsoft Store. Install the Store edition to get automatic updates and a clean install with no SmartScreen warning. This portable EXE (v1.4.0) is the final release delivered via GitHub; new features and fixes from here on ship through the Store edition. Store: https://apps.microsoft.com/detail/9NXX9L2ZW52W
- **Multi-language support (Korean/English)**: You can now choose the app language in Options — "Match system language (Auto)", "한국어", or "English". With Auto, any system language other than Korean falls back to English. A language change takes effect after you restart the program.
- **First-run usage notice**: The first time you launch the program, a notice explains that you're responsible for how you use it, that you must follow the terms of service of whatever program you automate, and that the app collects no personal data. You need to review and agree before you can start using the program.
- **Store edition storage locations**: The Store edition stores settings and logs in the standard system location, and macro files in a dedicated folder under Documents. The existing portable EXE version is unaffected and keeps storing everything inside its own program folder, exactly as before.

## Improvements

- **See every update at once in What's New**: The What's New window shown on launch now lists every change since the last version you checked, in one scrollable view. Up to 10 versions are shown; older history is available via the patch notes link at the bottom.
