# v1.5.2

## 개선 사항

- **다른 언어 화면에서 마우스 좌표 편집창 버튼 정렬 개선**: 마우스 동작 편집창에서 좌표 표 오른쪽의 버튼(캡처, 행 추가 등)이 좌표 표 위로 겹쳐 보이던 문제를 수정했습니다. 이제 어떤 언어(영어 등)에서도 창이 자동으로 알맞은 너비로 넓어져 버튼이 표를 가리거나 창 밖으로 잘리지 않습니다.
- **조건(IF) 편집창의 검색 영역 입력칸 정렬 개선**: 픽셀 색상·이미지 조건의 검색 영역(X/Y/W/H) 입력칸이 다른 언어 화면에서 마지막 칸이 창 밖으로 잘리던 문제를 수정했습니다.
- **변수 설정·연산 편집창 레이아웃 개선**: 변수 설정과 연산 편집창이 너무 작게 열려 [추가]·[삭제] 버튼이 표 위로 겹쳐 보이던 문제를 수정했습니다. 이제 창이 표와 버튼이 모두 보이는 크기로 열립니다.
