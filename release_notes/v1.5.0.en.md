# v1.5.0

## New Features

- **Split delay input into hours/minutes/seconds/ms**: The delay (time wait) step now has four separate fields for hours, minutes, seconds, and milliseconds. The tool waits for the sum of the values you enter, so you can set delays intuitively without converting them into milliseconds and can easily configure long waits of several minutes or more. Delay values in existing macros are preserved and replayed unchanged.

## Improvements

- **Stability and bug fixes**: The app now starts reliably even when its settings or favorites files are corrupted, and several issues found in step duplication, drag-and-drop, coordinate editing, background input, and image-search positioning have been fixed for better overall stability.
