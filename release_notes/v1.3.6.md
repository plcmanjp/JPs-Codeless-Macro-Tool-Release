# v1.3.6

## 개선 사항

- **안정성 보강**: 재생 중지/오류 시 눌린 키·마우스 버튼 OS 고착(release 짝 누락)을 방지하고, 실행 취소(Undo) 시 재생 중 변이를 막아 안정성을 강화했습니다.
- **예외 처리 및 가드**: LOOP_START count 비숫자 예외 안전 가드 및 hex/oct 문자열 변수 비교/변환 가드를 강화하였습니다.
- **클립보드 개선**: 동일 텍스트 재복사 시에도 sequence 번호를 우선적으로 확인하도록 개선하여 대기 타임아웃 문제를 해결했습니다.
- **UI 및 속도 개선**: OCR 미리보기 창에서 설정 초기화 시 신호 발생을 일시 중단시켜 로딩 속도를 최대 3배 이상 단축했습니다.
- **BOM 파일 인코딩**: 스크립트 파일을 열 때 UTF-8 BOM 인코딩 파일을 완벽히 지원하도록 변경하였습니다.
- **기능 무결성**: 좌표 캡처 카운트다운 도중 재클릭 시 발생하던 라벨 영구 고착 문제와 변수 모니터링 창의 업데이트 성능 낭비를 해결하였습니다.
