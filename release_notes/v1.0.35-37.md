# JP's Codeless Macro Tool v1.0.35~v1.0.37 업데이트

조건 체인(AND / OR / ELSE) 기능이 새로 도입되었습니다.

## 새로운 기능 (v1.0.35~v1.0.36)

- **AND 조건**: 앞 조건과 다음 조건이 모두 참일 때만 해당 스텝을 실행합니다.
- **OR 조건 (조건 결합자)**: 앞 조건이 거짓일 때 다음 조건을 이어서 평가합니다. 어느 조건이든 참이면 공통으로 실행할 스텝이 동작합니다.
- **ELSE**: 모든 조건이 거짓일 때 실행할 스텝을 지정할 수 있습니다.
- AND / OR / ELSE는 조건 시작(IF)과 같은 들여쓰기 레벨로 표시되며, 조건 끝(END) 하나로 전체 체인을 닫습니다.
- 샘플 스텝 조건문 탭에 AND / OR / ELSE를 활용하는 예제 5종이 추가되었습니다. (v1.0.36)

## 개선 사항 (v1.0.37)

- 녹화 중 단축키를 변경하면 새 키가 입력 제외 목록에 즉시 반영됩니다.
- 정규식 추출 스텝에 없는 변수명을 입력했을 때 경고가 표시됩니다.

## 버그 수정

- 이미지를 새로 캡처하거나 자르기를 적용한 뒤 화면 선택 영역 일부에 접근할 수 없던 문제를 수정했습니다. (v1.0.36)
- 중지 단축키를 눌러도 재생이 진행되지 않는 상태에서 재생 버튼이 비활성으로 남던 문제를 수정했습니다. (v1.0.37)
- 재생 완료 또는 중지 후 마지막으로 강조 표시된 행이 노란색으로 남던 문제를 수정했습니다. (v1.0.37)
- 저장에 실패했을 때 창이 그대로 닫혀 변경사항이 유실되던 문제를 수정했습니다. (v1.0.35)
- 주석 스텝을 더블클릭하거나 특정 좌표 버튼을 클릭할 때 앱이 종료되던 문제를 수정했습니다. (v1.0.35)
