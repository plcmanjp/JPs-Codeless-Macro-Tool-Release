# JP's Codeless Macro Tool v1.0.21 업데이트

출시일: 2026-05-14

## 변경 사항

- **프로그램 이름 변경**: 프로그램 이름이 `JP's Codeless Macro Tool`로, 실행 파일 이름이 `JPsCodelessMacroTool.exe`로 변경되었습니다.
  - 기존 `MacroFlow.exe`를 사용하던 버전에서 업데이트하는 경우 새 파일을 사용하면 됩니다.
  - 기존 매크로 제작, 편집, 녹화, 재생 기능은 그대로 사용할 수 있습니다.
  - 기존 프로젝트 파일도 그대로 열 수 있습니다.
