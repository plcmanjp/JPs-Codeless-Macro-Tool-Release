# JP's Codeless Macro Tool v1.1.0 업데이트

출시일: 2026-06-07

화면에 표시된 텍스트를 읽어 변수에 저장하고, 그 내용으로 조건 분기하거나 마우스를 정확한 위치로 이동시키는 OCR 기능이 추가되었습니다.

## 새로운 기능

- **OCR — 화면 텍스트 읽기 (OCR 읽기 스텝)**: 화면의 지정 영역(ROI)을 캡처해 텍스트를 인식하고 변수에 저장합니다. 재고 수량·상태 메시지·버튼 레이블 등 화면 텍스트를 매크로에서 직접 읽어 이후 스텝에 활용할 수 있습니다. 편집창의 "테스트(읽기)" 버튼으로 즉석 인식 결과와 미리보기를 확인할 수 있습니다.

- **OCR — IF 텍스트 포함 조건 (text_contains)**: IF 조건으로 "지정 영역 텍스트에 특정 키워드가 포함되어 있으면"을 설정할 수 있습니다. NOT 조건(포함되지 않으면)도 지원하며, 완전 일치 대신 포함 여부로 판단하기 때문에 OCR 오인식에 유연하게 대응합니다.

- **OCR — 인식 단어 위치 클릭 (텍스트 위치 사용)**: OCR 읽기 스텝에서 "텍스트 위치 저장" 옵션을 켜 두면, 이후 마우스 스텝에서 "텍스트 위치 사용" 옵션으로 인식된 단어의 화면 좌표를 바로 클릭 위치로 사용할 수 있습니다. 버튼 레이블 위치가 실행마다 달라지는 상황에 유용합니다.

- **OCR 미리보기 다이얼로그**: 편집창 테스트 버튼을 누르면 전처리된 이미지와 인식된 단어 박스 오버레이, 텍스트 목록을 한눈에 확인하는 미리보기 창이 열립니다. 업스케일 배율(1~5×)과 이진화(자동 Otsu / 수동 임계값) 옵션을 실시간으로 조정하며 최적 설정을 찾을 수 있습니다.

## 개선 사항

- **OCR 인식률 자동 개선 (업스케일)**: 작은 텍스트 영역을 캡처할 때 글자 높이 기준으로 배율을 자동 추정해 업스케일합니다. 기본 배율에서 인식되지 않던 작은 한글 폰트의 인식률이 크게 향상됩니다.

## 주의 사항

- OCR 기능은 Windows 언어팩이 필요합니다. 한국어 언어팩(ko-KR OCR 팩)이 설치되어 있지 않으면 OCR 스텝이 비활성화됩니다. Windows 설정 > 시간 및 언어 > 언어 > 한국어 > 선택적 언어 기능 > 기본 입력에서 OCR 팩을 설치하세요.
- 완전 일치 비교보다 텍스트 포함(contains) 판단을 권장합니다. OCR 특성상 글자 오인식이 발생할 수 있으므로, 핵심 키워드가 포함되는지 여부로 조건을 설계하세요.
- 2자 단어가 단독으로 줄에 배치된 경우 인식률이 낮을 수 있습니다. 인식 대상 영역에 주변 UI 요소가 함께 포함되도록 ROI를 잡으면 정확도가 개선됩니다.
