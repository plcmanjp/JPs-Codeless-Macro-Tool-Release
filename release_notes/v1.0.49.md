# JP's Codeless Macro Tool v1.0.49 업데이트

출시일: 2026-06-04

변수를 더 편리하게 사용할 수 있도록 선택형 변수(드롭박스)를 새로 도입하고, 변수 모니터의 편의성을 다방면으로 개선했습니다.

## 새로운 기능

- **선택형 변수 (드롭박스)**: 변수를 만들 때 후보값을 콤마로 나열해 지정하면(예: `A, B, C` 또는 `10, 20, 30`), 메인 화면 변수 설정 줄에 드롭박스가 생깁니다. 편집창을 따로 열지 않아도 드롭박스에서 값을 바로 골라 쓸 수 있으며, 선택한 값이 재생 시 그대로 키 입력·연산 등에 사용됩니다. 숫자 후보는 계산에, 글자 후보는 텍스트 입력에 활용할 수 있습니다.

## 개선 사항

- **변수 모니터에서 바로 편집**: 변수 모니터 창에서 변수를 더블클릭하면 해당 변수의 편집창이 바로 열립니다. 변수명·값 쪽을 더블클릭하면 변수 설정 편집으로, 설명 쪽을 더블클릭하면 변수 연산·정규식 편집으로 이어집니다.
- **변수 모니터 실시간 반영**: 변수를 추가·삭제·수정하거나 드롭박스에서 다른 값을 고르면 변수 모니터 목록과 설명이 즉시 갱신됩니다. 설명에는 선택값·초기값·연산식이 함께 표시됩니다.
- **변수 편집창 넓어짐**: 변수 설정·변수 연산 편집창을 넓혀 글자 잘림을 줄이고 한 화면에 더 많은 변수(약 7개)를 볼 수 있게 했습니다. 표에서 Delete 키로 선택한 줄을 바로 삭제할 수 있습니다.
- **설명 잘림 방지**: 메인 화면 변수 설명이 길어도 마우스를 올리면 전체 내용을 툴팁으로 확인할 수 있습니다.

## 버그 수정

- **변수 모니터 표시 정리**: 변수 모니터에 더 이상 쓰이지 않는 옛 설정 항목이 남아 보이던 문제를 정리했습니다.
