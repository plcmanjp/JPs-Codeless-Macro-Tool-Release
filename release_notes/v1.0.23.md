# JP's Codeless Macro Tool v1.0.23 업데이트

## 새로운 기능

- **샘플 스텝**: 자주 쓰는 기본 예제를 모아둔 샘플 스텝 기능을 추가했습니다.
  - 일반, 반복문, 조건문, 변수, 기타 5개 카테고리로 분류된 예제를 제공합니다.
  - 샘플 스텝은 읽기 전용이며, 더블클릭하거나 확인 버튼을 누르면 현재 위치에 바로 삽입됩니다.
  - 기존 기본 즐겨찾기 항목이 샘플 스텝으로 이관되어, 새로 시작하는 사용자는 빈 즐겨찾기로 시작합니다.
