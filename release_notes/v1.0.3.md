# JP's Codeless Macro Tool v1.0.3 업데이트

v1.0.1의 모든 기능을 포함하며, 변수 시스템이 추가되었습니다.

## 새로운 기능

- 변수 설정 기능을 추가했습니다. 반복 중 값을 바꿔 가며 텍스트를 생성하거나 동작을 제어할 수 있습니다.
- 실행 중 변수 값을 실시간으로 보여주는 변수 모니터 기능을 추가했습니다.
- 변수 값을 계산하는 변수 계산 기능을 추가했습니다.
- 반복문 안에서 변수를 사용할 수 있습니다.

## 개선 사항

- 마우스 랜덤 좌표 기능을 개선했습니다.
- 옵션 기본값 버튼과 단축키 구성을 정리했습니다.

## 버그 수정

- 한글 입력 관련 문제를 수정했습니다.
- 이미지 미리보기 갱신 문제를 수정했습니다.
