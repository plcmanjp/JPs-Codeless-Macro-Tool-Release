# JP's Codeless Macro Tool v1.2.2 업데이트

출시일: 2026-06-12

프로젝트 이동·공유 시 이미지 경로가 깨지던 문제를 해결하고, 즐겨찾기 정렬 등 여러 동작을 안정화한 업데이트입니다. 사용 방법은 이전과 동일합니다.

## 개선 사항

- **프로젝트 이동·공유 호환성**: 이미지 조건에 등록한 이미지(여러 장)와 무시영역 마스크의 경로를 프로젝트 폴더 기준으로 저장하도록 개선했습니다. 프로젝트 폴더를 옮기거나 다른 PC로 복사해도 이미지 검색이 정상 동작합니다. 이전 버전에서 만든 프로젝트도 열기만 하면 자동으로 경로가 복구됩니다.

- **즐겨찾기 순서 변경 안정화**: 즐겨찾기 관리 창에서 마우스 드래그로 순서를 바꾼 뒤 확인을 누르면, 화면에서 선택한 항목이 정확히 추가되고 바뀐 순서도 그대로 저장됩니다.

- **변수 표시 안정성**: 16진수·8진수 변수에 문자값이 들어간 상태로 텍스트 입력에 사용해도 매크로가 중단되지 않고 원문 그대로 입력되도록 개선했습니다.

- **변수 계산(CALC) 보호 강화**: 지나치게 큰 거듭제곱 수식이 프로그램을 멈추게 하던 문제를 차단했습니다. 일반적인 수식은 모두 이전과 동일하게 동작합니다.

- **프로젝트 이름 검사**: 새 프로젝트 만들기·다른 이름으로 저장·이름바꾸기에서 폴더 이름에 쓸 수 없는 문자를 입력하면 미리 안내합니다.

- **종료 안정성**: 매크로 재생 중에 프로그램을 닫아도 안전하게 종료되도록 개선했습니다.
