# JP's Codeless Macro Tool v1.0.42 업데이트

## 새로운 기능

- **녹화 마우스 드래그 개선**: 클릭 후 드래그를 이동 경로(궤적)와 이동 시간까지 정확히 녹화하고 재생합니다.
  - 드래그를 "누른 채 이동"으로 기록해, 재생 시 그림판 등에서 드래그 선이 그대로 재현됩니다.
  - 마우스가 가만히 있던 시간은 딜레이로, 실제로 움직인 시간은 이동 시간으로 구분해 기록합니다.
  - 마우스 이동·클릭·누름·뗌 스텝 설명에 이동 시간이 함께 표시됩니다.

## 개선 사항

- IF / ELSE IF 조건 체인의 각 조건절에서 AND, OR, ELSE IF, ELSE 추가 가능 여부를 바로잡았습니다. 두 번째 이후 조건절에서도 논리 스텝을 추가할 수 있습니다.

## 버그 수정

- 녹화 시 첫 마우스 이동 좌표의 이동 시간이 비정상적으로 크게 기록되던 문제를 수정했습니다. 이제 첫 좌표의 이동 시간은 0입니다.
