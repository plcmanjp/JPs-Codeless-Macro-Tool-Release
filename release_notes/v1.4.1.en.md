# v1.4.1

## Improvements

- **What's New works offline**: The What's New window now appears without an internet connection. The change history is bundled with the program, so you can review every change since the version you last checked in one scrollable view, even offline.
- **Updates handled automatically by the Microsoft Store**: Update checks are now fully delegated to the Microsoft Store's automatic updates. The Store installs the latest version for you without a separate update prompt.
- **Open Log Folder**: Added an "Open Log Folder" button in the options dialog. Clicking it immediately opens the current log folder (LocalAppData folder for the Store version, or the 'logs' subfolder for the portable version) in File Explorer.
