# JP's Codeless Macro Tool v1.0.46 업데이트

v1.0.45의 개선 사항을 모두 포함하며, 좌표 자동생성 스네이크 모드가 추가되었습니다.

## 새로운 기능

- **좌표 자동생성 스네이크(왕복) 모드**: 좌표 자동생성 화면에 "생성 모드" 선택 항목이 추가되었습니다.
  - **지그재그**: 기존 방식입니다. 모든 행이 같은 방향으로 좌표를 채웁니다(기본값).
  - **스네이크**: 왕복 방식입니다. 홀수 행과 짝수 행의 방향이 번갈아 바뀝니다. 화면에서 지그재그보다 자연스러운 순서로 좌표를 순회할 때 유용합니다. 1줄(1D) 배열에서는 지그재그와 동일하게 동작합니다.

## 개선 사항

- **편집 단축키 옵션화**: 좌표 캡처(기본 F2)와 보조좌표(기본 F3) 단축키를 옵션 창의 "편집 단축키" 그룹에서 직접 변경할 수 있습니다.
- **보조좌표 단축키(F3)**: 좌표 입력 테이블에서 F3으로 행을 빠르게 추가하거나, 좌표 자동생성 화면에서 F3을 눌러 끝점 피치 계산을 즉시 시작할 수 있습니다.
- **좌표 테이블 버튼**: 버튼 텍스트에 단축키가 표시되고, 버튼 크기가 통일되었습니다.
- **좌표 테이블 다중 선택**: Ctrl+A·Shift/Ctrl+클릭·Shift+↑↓로 여러 행을 동시에 선택하고 한 번에 삭제하거나 이동할 수 있습니다.
- **옵션 토글버튼**: "다중 실행 허용"과 "항상 위" 옵션이 ON/OFF 토글 버튼으로 바뀌었습니다.
- **옵션 창 2열 배치**: 옵션 항목이 2열로 나뉘어 창의 세로 길이가 줄었습니다.

## 버그 수정

- 마우스 액션 편집 도중 다른 스텝으로 전환했을 때 화면에 좌표 미리보기 표시가 남던 문제를 수정했습니다.
- 재생 후 잔류한 IF 조건 결과 표시가 편집 모드에서 행 추가·삭제·이동 시 엉뚱한 행으로 전이되던 오동작을 수정했습니다. 보기→편집 모드 전환 시 결과 표시가 자동으로 정리됩니다.
