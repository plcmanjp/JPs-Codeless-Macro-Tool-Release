# JP's Codeless Macro Tool v1.0.39 업데이트

## 새로운 기능

- **조건 탈출 (BREAK)**: 중첩된 IF 블록 안에서 BREAK 스텝을 실행하면 블록을 한 번에 빠져나옵니다.
  - 반복(LOOP) 경계에서는 다음 반복이 계속 진행됩니다.
  - 포인터(POINTER) 경계에서는 포인터 호출 이후로 실행이 복귀됩니다.
  - 조건 블록 바깥에서 사용하면 아무 효과가 없습니다.
  - 컨텍스트 메뉴 → 조건 논리 → BREAK (조건 탈출)에서 추가할 수 있습니다.
- **항상 참 / 항상 거짓 조건**: 단순 분기 테스트나 IF/ELSE 흐름 작성 시 사용할 수 있는 조건을 추가했습니다.

## 개선 사항

- 이미지 조건에서 이미지 경로가 비어있거나 파일이 없으면 해당 IF 블록 전체를 건너뜁니다. 두 조건(이미지 발견됨 / 이미지 없음) 모두 동일하게 처리됩니다.
- 비정상적인 조건 체인 구조가 있는 매크로를 열면 1회 안내 메시지가 표시됩니다.
