# v1.3.5

## 개선 사항

- **데이터 보호**: 변경된 매크로에서 "새로 만들기"나 저장을 할 때 저장이 실패하거나 이름 입력을 취소하면 작성 중인 스텝이 사라지던 문제를 수정했습니다. 이제 저장이 끝나지 않으면 기존 작업이 그대로 유지됩니다.
- **변경 표시 정확도**: 예전 버전에서 만든 조건(IF) 매크로를 열었을 때 아무것도 바꾸지 않았는데 제목에 변경 표시(*)가 뜨던 문제를 수정했습니다.
- **다중 모니터 좌표**: 모니터를 여러 대 쓰거나 고해상도 환경에서 좌표를 캡처할 때 값이 잘못 잘려 저장되던 문제를 수정했습니다.
- **즐겨찾기**: 즐겨찾기 파일 일부가 손상되어도 나머지 항목을 보존하도록 했고, 관리 창에서 항목을 삽입하지 않고 정렬·삭제만 저장할 수 있는 "저장만" 버튼을 추가했습니다.
- **안정성 보강**: 재생 중 변수 모니터 조작, OCR 미리보기 빠른 닫기, 화면 캡처·녹화·로그 등 여러 기능의 예외 처리와 안정성을 폭넓게 보완했습니다.
