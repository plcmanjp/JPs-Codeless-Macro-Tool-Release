# JP's Codeless Macro Tool v1.2.0 업데이트

출시일: 2026-06-08

스텝을 추가할 때 편집창을 열고 종류를 다시 고르는 2단계 과정을 없애고, 메뉴에서 원하는 동작을 바로 선택해 추가할 수 있도록 개선했습니다.

## 새로운 기능

- **스텝 추가가 한 번에**: 하단 "➕ 스텝 추가" 버튼을 클릭하거나 스텝 목록을 우클릭하면 키보드·마우스·텍스트 입력·변수·딜레이·포인터 등 추가할 동작 종류를 메뉴에서 바로 선택할 수 있습니다. 기존에는 편집창을 열고 종류 드롭박스를 다시 선택해야 했지만, 이제 메뉴 한 번으로 끝납니다.

- **조건(IF)도 메뉴에서 바로 선택**: "◆ 조건(IF)" 항목에서 픽셀 색상·이미지 인식·변수 비교·화면 텍스트 포함·키 눌림 상태 등 8가지 조건 종류를 메뉴에서 바로 골라 추가할 수 있습니다. 선택한 조건이 편집창의 초기값으로 설정되어 열리며, 열린 뒤에도 조건 종류를 변경할 수 있습니다.

- **반복·포인터도 메뉴에서 바로**: 반복 블록(LOOP)과 포인터 시작·포인터 호출도 같은 스텝 추가 메뉴에서 선택할 수 있습니다. 기능별 전용 버튼은 그대로 유지됩니다.

## 개선 사항

- **블록 구조 자동 보정**: IF 블록이나 반복 블록을 추가한 뒤 편집창에서 동작 종류를 다른 것으로 바꿔도 블록 구조(시작·끝 쌍)가 어긋나지 않도록 자동으로 처리합니다. 조건 블록의 AND·OR·ELSE IF 스텝은 편집창에서 종류 변경이 잠겨 구조가 깨지는 실수를 원천 방지합니다.
