# v1.5.2

## Improvements

- **Better button alignment in the coordinate editor for other languages**: In the mouse-action edit dialog, the buttons to the right of the coordinate table (Capture, Add row, etc.) could overlap the table. The dialog now automatically widens to fit, so the buttons never cover the table or get clipped off-screen in any language (including English).
- **Better search-region input alignment in the condition (IF) editor**: For pixel-color and image conditions, the search region (X/Y/W/H) inputs could have the last box clipped off-screen in non-Korean languages. This is now fixed.
- **Improved layout of the variable-set and calculation editors**: These editors opened too small, causing the Add/Delete buttons to overlap the table. They now open large enough to show both the table and the buttons.
