# JP's Codeless Macro Tool v1.0.34 업데이트

출시일: 2026-05-16

## 새로운 기능

- **단일 실행 잠금**: 기본적으로 프로그램을 동시에 여러 개 실행할 수 없도록 잠금이 적용됩니다.
- **다중 실행 허용 옵션**: 옵션 창에서 "다중 실행 허용"을 켜면 다음 실행부터 여러 개를 동시에 열 수 있습니다.

## 개선 사항

- 변수 바꾸기 스텝의 원본 변수와 저장 변수 선택을 드롭박스로 통일했습니다. 선언된 변수가 없어도 클립보드를 선택할 수 있습니다.
