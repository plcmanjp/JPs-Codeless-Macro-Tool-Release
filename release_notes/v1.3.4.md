# JP's Codeless Macro Tool v1.3.4 업데이트

출시일: 2026-06-16

키 입력 관련 스텝을 추가할 때 더 편하고 실수 없이 쓸 수 있도록 다듬은 업데이트입니다.

## 개선 사항

- **키 기본 선택 제거**: 키 입력 스텝이나 "키 눌림" 조건을 새로 추가할 때 'a' 키가 자동으로 선택되어 있던 동작을 없앴습니다. 이제 아무 키도 선택되지 않은 상태로 시작하므로, 원하는 키만 직접 골라 넣을 수 있습니다.
- **빈 키 저장 방지**: 키를 하나도 선택하지 않은 채로는 스텝이 저장되지 않도록 안내합니다. 실수로 빈 키 스텝이 만들어지는 일을 막아 줍니다. (기존에 빈 키로 저장된 스텝이 있어도 재생 시 오류 없이 자연스럽게 건너뜁니다.)
