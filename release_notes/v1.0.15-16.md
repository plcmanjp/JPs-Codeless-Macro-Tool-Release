# JP's Codeless Macro Tool v1.0.15~v1.0.16 업데이트

## 새로운 기능

- **키보드 조합키 지원** (v1.0.16): `Ctrl+C`, `Ctrl+V`, `Shift+C`, `Ctrl+Alt+Delete` 같은 여러 키를 함께 누르는 조합키를 스텝으로 만들 수 있습니다.

## 개선 사항 (v1.0.15)

- 즐겨찾기에 같은 이름을 추가하려 할 때 중복 여부를 알려줍니다. 기존 항목을 덮어쓸지, 다른 이름으로 다시 입력할지 선택할 수 있습니다.
