# JP's Codeless Macro Tool v1.0.32 업데이트

출시일: 2026-05-16

## 새로운 기능

- **글로벌 딜레이**: 실행 스텝 사이에 암묵적으로 적용되는 대기 시간을 설정할 수 있습니다.
  - 기본값은 10ms이며, 1~10,000ms 범위에서 설정할 수 있습니다.
  - 대상 프로그램이 입력을 놓치는 경우 값을 늘려 더 안정적으로 실행할 수 있습니다.
  - 명시적 딜레이 스텝 뒤에는 글로벌 딜레이가 중복 적용되지 않습니다.
- **샘플 스텝 보강**: 마우스 액션 탭과 키보드 액션 탭이 추가되었습니다. 반복문 탭에 구구단 예제가 추가되었습니다.
- 프로그램이 비활성 상태일 때는 F2(좌표 등록), F5(녹화창 열기), 세미콜론(주석 추가) 같은 편집 단축키가 동작하지 않도록 제한했습니다. 재생·녹화 관련 단축키는 비활성 상태에서도 계속 사용할 수 있습니다.
