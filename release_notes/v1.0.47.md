# JP's Codeless Macro Tool v1.0.47 업데이트

출시일: 2026-06-03

이미지 검색 기능이 더 다양한 화면 환경에서 동작할 수 있도록 두 가지 옵션이 추가되었습니다.

## 새로운 기능

- **이미지 검색 — 컬러 무시(그레이스케일)**: 이미지 캡처·편집 창의 "이미지 필터링" 그룹에 "컬러 무시(그레이스케일)" 체크박스가 추가되었습니다. 이 옵션을 켜면 화면과 템플릿 이미지를 모두 흑백으로 변환한 뒤 비교합니다. 테마 변경, 야간 모드, 배경 색상이 자주 바뀌는 환경에서 형태만 맞으면 찾을 수 있어 유용합니다. 옵션은 이미지마다 개별로 설정할 수 있습니다.

- **이미지 검색 — 무시 영역(자유펜 마스크)**: 이미지 캡처·편집 창에 "✏ 무시영역 펜" 기능이 추가되었습니다. 이미지에서 매번 바뀌는 부분(숫자, 상태 아이콘 등)을 펜으로 직접 칠하면 그 영역이 매칭 대상에서 제외됩니다.
  - 좌클릭으로 마스크를 그리고, 우클릭으로 지웁니다.
  - 마우스 휠을 돌려 펜 크기를 조절할 수 있습니다(2~60px).
  - 마스크를 전체 초기화하려면 "마스크 지우기" 버튼을 누릅니다.
  - 마스크는 이미지별로 저장됩니다. 이미지를 자르기해도 해당 영역의 마스크는 그대로 유지됩니다.

## 개선 사항

- 업데이트 알림이 버전 매니페스트 기반으로 더 정확하게 동작합니다. 새 버전이 있을 때 알림이 표시되며, "패치노트를 확인하시겠습니까?" 메시지를 통해 해당 버전의 패치노트 페이지로 바로 이동할 수 있습니다.
