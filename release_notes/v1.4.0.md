# v1.4.0

## 새로운 기능

- **Microsoft Store 정식 출시 — 이후 업데이트는 스토어에서 받으세요**: JP's Codeless Macro Tool이 Microsoft Store에 정식 출시되었습니다. 스토어판을 설치하면 앞으로 업데이트가 자동으로 이루어지고, SmartScreen 경고 없이 설치됩니다. 이 포터블 EXE(v1.4.0)가 GitHub를 통한 마지막 배포이며, 이후 새로운 기능·수정은 스토어판에서 제공됩니다. 스토어: https://apps.microsoft.com/detail/9NXX9L2ZW52W
- **다국어 지원(한국어/영어)**: 옵션에서 사용할 언어를 고를 수 있습니다. "시스템 언어에 맞춤(자동)"·"한국어"·"English" 중에서 선택하며, 자동으로 두면 한국어가 아닌 시스템에서는 영어로 표시됩니다. 언어를 바꾼 뒤에는 프로그램을 다시 시작해야 적용됩니다.
- **첫 실행 사용 안내**: 프로그램을 처음 실행하면 사용 책임, 대상 프로그램의 이용약관 준수, 개인정보 미수집 등을 안내하는 화면이 뜹니다. 내용을 확인하고 동의해야 프로그램을 사용할 수 있습니다.
- **스토어판 저장 위치**: 스토어판은 설정·로그를 시스템 표준 위치에, 매크로 파일을 문서 폴더의 전용 폴더에 저장합니다. 기존 포터블 EXE 버전은 지금처럼 프로그램 폴더 안에 그대로 저장되며 달라지는 점이 없습니다.

## 개선 사항

- **업데이트 내역(What's New) 한 번에 확인**: 프로그램을 열 때 뜨는 업데이트 내역 창에서, 마지막으로 확인한 버전 이후의 모든 변경사항을 스크롤로 한 번에 볼 수 있습니다. 최대 10개 버전까지 표시하며, 그보다 오래된 내역은 화면 하단의 패치노트 링크에서 확인할 수 있습니다.
