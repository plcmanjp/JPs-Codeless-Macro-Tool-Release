# JP's Codeless Macro Tool v1.0.14 업데이트

## 새로운 기능

- **변수 기능 재정리**: 변수 설정과 변수 계산 기능을 더 명확하게 분리했습니다.
- 텍스트 입력 스텝에서 `%변수명` 형식으로 변수를 쉽게 넣을 수 있습니다. 예: `%카운터`를 입력하면 실행 시 해당 변수 값으로 바뀝니다.
- 변수 계산에서 `++`, `--` 같은 간단한 증가/감소 표기를 지원합니다.

## 버그 수정

- 변수 이름이 비슷할 때(예: `%카` / `%카운터`) 짧은 이름이 먼저 치환되던 문제를 수정했습니다.
