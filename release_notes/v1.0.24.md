# JP's Codeless Macro Tool v1.0.24 업데이트

## 새로운 기능

- **보기 모드 / 편집 모드 분리**: 스텝 목록에 보기 모드와 편집 모드가 도입되었습니다.
  - 프로그램 실행 시 기본값은 보기 모드로, 실수로 스텝이 바뀌는 일을 줄여 줍니다.
  - 보기 모드에서 스텝 추가·편집·삭제 등 변경 작업을 시도하면 편집 모드 전환 확인창이 나타납니다.
  - 재생 시작 시 자동으로 보기 모드로 전환됩니다.
  - 상단에서 보기/편집 모드를 직접 전환할 수 있고, 현재 모드가 상태바에 표시됩니다.
