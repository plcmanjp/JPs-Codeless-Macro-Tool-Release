# v1.4.1

## 개선 사항

- **오프라인에서도 업데이트 내역 확인**: 업데이트 내역(What's New) 창이 인터넷 연결 없이도 표시됩니다. 변경 내역이 프로그램에 함께 포함되어, 오프라인 환경에서도 마지막으로 확인한 버전 이후의 모든 변경사항을 스크롤로 볼 수 있습니다.
- **업데이트는 Microsoft Store에서 자동으로**: 업데이트 확인이 Microsoft Store 자동 업데이트로 완전히 전환되었습니다. 별도의 업데이트 알림 창 없이 스토어가 최신 버전을 자동으로 설치합니다.
- **로그 폴더 열기**: 옵션 설정 화면에 "로그 폴더 열기" 버튼이 추가되었습니다. 클릭 시 현재 실행 모드에 맞는 실제 로그 폴더(스토어 버전의 경우 LocalAppData 폴더, 포터블 버전의 경우 프로그램 폴더 내 logs)가 파일 탐색기로 바로 열립니다.
