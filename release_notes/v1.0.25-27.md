# JP's Codeless Macro Tool v1.0.25~v1.0.27 업데이트

## 새로운 기능

- **변수 계산 결과를 클립보드로** (v1.0.25): 변수 계산 결과를 일반 변수 외에 클립보드로도 보낼 수 있습니다.
- **변수 바꾸기 개선** (v1.0.26): 변수 바꾸기 스텝에서 원본 변수와 저장 대상 변수를 따로 선택할 수 있습니다. 클립보드를 원본이나 저장 대상으로 사용할 수 있어 복사한 문자열을 바로 가공할 수 있습니다.

## 개선 사항

- **스텝 목록 들여쓰기 개선** (v1.0.27): 반복문, 조건문, 포인터처럼 블록이 깊어지는 스텝을 더 보기 쉽게 표시합니다.
