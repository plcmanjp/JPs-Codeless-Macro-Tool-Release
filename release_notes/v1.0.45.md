# JP's Codeless Macro Tool v1.0.45 업데이트

## 새로운 기능

- **편집 단축키 옵션화**: 좌표 캡처(기본 F2)와 보조좌표(기본 F3) 단축키를 옵션 창의 "편집 단축키" 그룹에서 직접 변경할 수 있습니다. 기존에는 F2·F3으로 고정되어 있었습니다.
- **보조좌표 단축키(F3)**: 마우스 액션 좌표 입력 테이블에서 F3으로 행을 빠르게 추가하거나, 좌표 자동생성 화면에서 F3을 눌러 화면에서 끝점을 클릭하는 피치 계산을 즉시 시작할 수 있습니다.

## 개선 사항

- **좌표 테이블 버튼**: 캡처·행 추가·삭제·이동 버튼 텍스트에 단축키가 함께 표시되고, 버튼 크기가 통일되어 보기 편해졌습니다.
- **좌표 테이블 다중 선택**: 좌표 행을 Ctrl+A(전체 선택), Shift·Ctrl+클릭, Shift+↑↓로 여러 행을 동시에 선택할 수 있습니다. 선택한 여러 행을 한 번에 삭제하거나 위아래로 이동할 수 있습니다.
- **옵션 토글버튼**: "다중 실행 허용"과 "항상 위" 옵션이 드롭다운 선택에서 ON/OFF 토글 버튼으로 바뀌어 현재 상태를 한눈에 확인하고 빠르게 전환할 수 있습니다.
- **옵션 창 2열 배치**: 옵션 항목이 2열로 나뉘어 창의 세로 길이가 줄었습니다.

## 버그 수정

- 마우스 액션을 편집하는 도중 다른 스텝으로 전환했을 때 화면에 좌표 미리보기 표시가 남던 문제를 수정했습니다.
