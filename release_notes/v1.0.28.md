# JP's Codeless Macro Tool v1.0.28 업데이트

## 새로운 기능

- **마우스 좌표 어레이**: 마우스 이동, 클릭, 누름 스텝에 여러 좌표를 목록으로 등록할 수 있습니다.
  - 반복문 안에서 좌표 어레이가 있는 마우스 스텝을 실행하면, 반복 횟수에 따라 좌표가 순서대로 사용됩니다.
  - 좌표 개수보다 반복 횟수가 많으면 첫 좌표부터 다시 순환합니다.
  - 좌표 목록 편집 창에서 F2로 현재 마우스 위치를 바로 추가하고, 행 이동·편집·삭제를 할 수 있습니다.
