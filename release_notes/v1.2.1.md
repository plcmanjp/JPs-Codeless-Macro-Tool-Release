# JP's Codeless Macro Tool v1.2.1 업데이트

출시일: 2026-06-12

프로그램의 보안과 안정성을 강화하고, 변수 입력 편의와 여러 동작을 개선한 업데이트입니다. 사용 방법은 이전과 동일하며, 내부 동작이 더 안전하고 안정적으로 개선되었습니다.

## 개선 사항

- **보안 강화**: 업데이트 확인 시 통신 인증서 검증을 복원하고, 변수 계산(CALC) 수식 입력의 안전성을 강화했습니다. 외부에서 받은 매크로 파일을 열어 실행할 때도 안전하게 동작합니다.

- **변수 입력 편의 개선**: 변수 계산(CALC) 수식 칸에서 `%`를 입력하면 변수 선택 목록이 즉시 나타납니다. 정규식 추출·OCR·텍스트 조건의 변수 입력을 드롭박스로 정리해 선언한 변수를 쉽게 고를 수 있습니다. 이미지 조건(이미지 있음/없음) 편집 창의 레이아웃 표시도 정돈했습니다.

- **비정상 종료 후 재실행 개선**: 프로그램이 강제 종료되거나 예기치 않게 닫힌 뒤 다시 실행할 때, 이전 실행 흔적에 막히지 않고 정상적으로 켜지도록 개선했습니다.

- **반복 재생 안정성**: 매크로를 여러 번 반복 재생할 때 누적되던 메모리 사용을 정리해 장시간 실행에서도 안정적으로 동작합니다.

- **화면 텍스트 읽기(OCR) 안정성**: 텍스트 인식이 지연될 때 최대 10초 후 자동으로 넘어가도록 타임아웃을 추가했고, 편집 창의 인식 미리보기 중에도 화면이 멈추지 않도록 개선했습니다.

- **그 외 버그 수정**: 즐겨찾기 파일의 인코딩 호환성, 이미지 검색 임시 파일 정리, 보조 창(변수·로그 모니터)의 '항상 위' 표시 동작을 개선했습니다.
