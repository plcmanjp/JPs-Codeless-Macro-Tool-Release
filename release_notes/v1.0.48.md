# JP's Codeless Macro Tool v1.0.48 업데이트

출시일: 2026-06-03

변경 내용 안내 표시 방식이 개선되었습니다. 사용자에게 보이는 조작 기능 변화는 없으며, 내부 동작 방식을 더 유연하게 다듬은 소규모 릴리스입니다.

## 새로운 기능

- **변경 내용 안내가 항상 최신으로 표시**: 이전에는 변경 내용 안내가 프로그램 안에 포함되어 있어, 안내를 갱신하려면 프로그램을 새로 받아야 했습니다. 이제 인터넷에 연결된 환경에서는 변경 내용 안내를 온라인에서 직접 불러와 표시합니다. 프로그램을 업데이트하지 않아도 최신 안내를 볼 수 있습니다.

## 개선 사항

- 인터넷이 연결되지 않은 환경에서는 변경 내용 안내 팝업이 표시되지 않습니다. 프로그램의 매크로 기능 자체는 오프라인에서도 정상 동작합니다.
