# v1.5.1

## 개선 사항

- **백그라운드 재생 중 대상 창 종료 감지**: 백그라운드 입력으로 매크로를 실행하는 중 대상 창이 닫히면 이제 명확하게 알림을 띄우고 재생을 멈춥니다. 이전에는 알림 없이 그냥 지나칠 수 있었습니다.
- **화면 캡처·화면 텍스트 인식(OCR)·이미지 검색 안정성 개선**: 화면 캡처, OCR 미리보기, 이미지 검색 과정의 오류 상황 처리와 응답성을 개선했습니다.
- **단축키 중복 배정 경고**: 같은 단축키를 서로 다른 두 기능에 배정하면 경고로 알려 실수로 인한 단축키 충돌을 막습니다.
- **재생 엔진 응답 없음 상황 안내**: 재생 엔진이 드물게 응답하지 않는 상황이 발생하면 앱을 재시작하도록 안내합니다.

## 버그 수정

- 즐겨찾기 파일이 손상된 경우에도 앱이 정상적으로 시작되도록 수정했습니다. (이전에는 시작이 막힐 수 있었습니다.)
- 녹화 시작·정지, 재생 정지 시의 동작을 더 안정적으로 다듬었습니다.
- 마지막에 열었던 작업 파일을 자동으로 불러오는 과정에서 문제가 생겨도 원본 파일이 손상되지 않도록 보호했습니다.
- 여러 창을 동시에 열어 두고 저장할 때 파일이 서로 충돌해 깨지던 문제를 방지했습니다.
