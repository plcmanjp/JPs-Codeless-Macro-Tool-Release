# JP's Codeless Macro Tool v1.0.33 업데이트

## 새로운 기능

- **변수 바꾸기 다중 치환**: 변수 바꾸기 스텝에서 여러 개의 검색/치환 규칙을 한 번에 등록할 수 있습니다.
  - 치환 규칙은 테이블 위에서 아래 순서대로 적용됩니다.
  - 규칙 행은 추가·삭제·위아래 이동 버튼으로 관리할 수 있고, `Delete` 키로도 삭제됩니다.
  - 기존 단일 치환 형식으로 저장된 프로젝트는 열면 자동으로 첫 번째 치환 행으로 읽어 그대로 사용할 수 있습니다.

## 개선 사항

- 변수 바꾸기의 검색 문자와 바꿀 문자 입력란에서 `%`를 입력하면 선언된 변수 이름을 고를 수 있는 팝업이 나타납니다. 변수 계산, 조건 비교, 정규식 추출 입력란에서도 동일하게 동작합니다.
